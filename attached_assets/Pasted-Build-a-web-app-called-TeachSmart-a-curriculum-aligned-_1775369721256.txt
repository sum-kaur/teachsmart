Build a web app called TeachSmart — a curriculum-aligned resource finder for Australian teachers. Use Node.js + Express backend with a single public/index.html frontend.

1. Dashboard — sidebar nav, two CTA cards (Generate Resource, Adapt File), recent resources table

3. Results — alignment score bar showing percentage + syllabus + outcome codes; filter panel with toggles; resource cards with trust badges (verified/bias checked), a "Why this?" teal callout, local context tags, and an "Adapt for Year 9" button
4. Lesson Plan — resource reference header, lesson overview with activities timeline, a blue NSW local context callout box, differentiated questions with Foundation/Core/Extension pills, teacher notes textarea, Export PDF and Google Classroom buttons
