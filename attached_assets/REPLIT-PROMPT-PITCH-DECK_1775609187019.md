```
│     Finds AC v9        (AIATSIS, NMA,   │

│                                          │
│  3. Merge & Rank by Trust + Relevance   │
│                                          │
└──────────────────────────────────────────┘

+ Trust Scorecard
+ Local Lens tip
+ Lesson plan ready
```

Stream three: we merge the results, rank by relevance and trust score, and inject local context based on Sarah's postcode. All in under 3 seconds. ## Deck-Wide Design Notes

- **Font:** Use a clean, readable sans-serif (e.g., Inter, Poppins)
- **Colours:** Teal (primary), slate (secondary), white (background), red/green for comparisons
- **Logo:** Consistent placement (usually top-left or bottom-right)
- **Transitions:** Minimal, professional (no spinning or flashy effects)
- **Animations:** Subtle entrance animations on bullets; nothing distracting
- **Images:** High quality, Australian context where possible
- **Data sources:** Cite them (AC v9 from ACARA, CurricuLLM benchmarks from curricullm.com)
- **Accessibility:** High contrast, readable text sizes (18pt minimum for body text)

---

## Presentation Tips

1. **Rehearse to time:** This is a 5-minute pitch. Every second counts. Rehearse 3–5 times.
2. **Pace:** Speak slowly. Let information land. Don't rush.
3. **Eye contact:** Look at the judges, not the slides.
4. **Confidence:** You've built something real. Own it.
5. **Demo:** If live demo, have a backup (video of the demo). Always prepare for technical failure.
6. **Story beats:** Problem → Persona → Solution → Demo → Why Different → Impact → Call to Action
7. **Tone:** Professional but warm. This is about helping teachers, not just selling tech.
8. **Questions:** Prepare for "How is this different from Scootle?" and "What about teacher privacy?" and "Can this scale internationally?"

## Expected Judge Questions

1. **"How is this different from [Competitor]?"** → "We're the only one combining CurricuLLM accuracy with trust verification and local contextualisation. We also don't replace teacher judgment; we support it."

2. **"What about privacy?"** → "All searches are logged securely, no student data collected, no data sold. Teachers control what's shared."

3. **"How do you get teachers to adopt this?"** → "Free soft launch, partnership with state education departments, teacher referral network, word-of-mouth in professional learning communities."

4. **"What if your AI is wrong?"** → "CurricuLLM is 89% accurate. When it's unsure, we flag it in Tier B. Teachers always have the final say on curriculum alignment."

5. **"Can this work internationally?"** → "Yes. The architecture works for any country with a published curriculum in JSON format. We're starting with Australia; NZ, Singapore, UK next."
